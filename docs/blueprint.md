# **App Name**: MockInterviewPro

## Core Features:

- Job Description Upload: Allow users to upload a job description (e.g., PDF, TXT, DOCX).
- Resume Upload: Enable users to upload their resume for tailored interview preparation.
- Personalized Interview Simulation: Utilize the Gemini API to conduct mock interviews based on the uploaded job description and resume. The tool uses the resume and job description to provide helpful interview practice.
- Real-time Feedback: Provide instant feedback during the mock interview based on the responses given.
- Detailed Post-Interview Analysis: Generate a comprehensive report highlighting strengths, weaknesses, and areas for improvement after the mock interview, based on the AI's analysis.
- Performance History: Log history of the previous interview analysis to enable tracking progress of each performance

## Style Guidelines:

- Primary color: Deep blue (#293B5F) to convey professionalism and trust.
- Background color: Light gray (#F0F4F8) for a clean and modern look.
- Accent color: Teal (#3D948C) for highlights and call-to-action buttons to stand out from the calm background.
- Body and headline font: 'Inter', a sans-serif, provides a clean and modern reading experience.
